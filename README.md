# Personal-Asset-Tracker
Programatically tracks net worth and sets up alerts, etc.


# Setup
.env file needs to contain the following:
```
    POSTGRES_DB_1_USER = {pwd}
    POSTGRES_DB_1_PASSWORD = {pwd}
    POSTGRES_DB_1_DB = {pwd}

    POSTGRES_DB_2_USER = {pwd}
    POSTGRES_DB_2_PASSWORD = {pwd}
    POSTGRES_DB_2_DB = {pwd}
```
